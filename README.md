# Frontend Test API Project

This project demonstrates the design, development, and deployment of a minimal API built in C#. Frontend Test API is a valuable tool in web development, providing a reliable backend to test front end applications against during the learning and development process.

The API includes realistic sample data and is fully documented for ease of use. It incorporates health checks, and simulates real world conditions such as network slowdowns and error responses, allowing developers to build and test more resilient front end applications.

The project is deployed in two environments: as a Docker container and as a web application hosted on a virtual private server, demonstrating practical experience with containerization and cloud deployment workflows.

The finished API serves as a reusable testing tool for front end development and showcases skills in API design, backend development, deployment, and documentation using tools such as Scalar for API reference pages.

## Tools Used

- C#
- .NET Minimal API
- Scalar (API documentation)
- Docker
- Virtual Private Server (VPS) deployment
- Health checks
